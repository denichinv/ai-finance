# 💸 AI Finance Dashboard

A full-stack finance dashboard built with **React, TypeScript, and ASP.NET Core**, focused on real-world architecture and API integration.

---

## 🚀 Overview

This project is being developed in collaboration with a backend developer to simulate a real-world product workflow.

The application is structured as a **monorepo**, where both frontend and backend live in a single codebase.


---

## ✨ Features

- 🔁 Full CRUD for transactions (create, read, delete)
- 🔗 Frontend ↔ Backend integration (REST API)
- 📊 Financial insights and category tracking
- 🎬 UI animations with Framer Motion
- 🧩 Modular component-based architecture

---

## 🧱 Tech Stack

**Frontend**
- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Axios

**Backend**
- ASP.NET Core Web API (C#)
- Entity Framework Core
- SQLite


---

## 🏗️ Architecture

- 📦 Monorepo structure (frontend + backend in one project)
- 🔄 Shared development workflow across frontend and backend
